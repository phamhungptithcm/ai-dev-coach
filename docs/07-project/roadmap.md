# Product Roadmap

## V1: Habit Coaching Extension (Current)

- profile onboarding (role, skill, habit goal)
- required prompt-builder form
- real-time coaching overlays
- large paste warning system
- local dependency tracking

## V2: Learning Analytics and Better Guidance

- session summaries and progress trends
- per-platform tuning for warning quality
- customizable habit frameworks by role

## V3: IDE Companion

- VS Code integration
- pull request coaching hooks
- debugging-first prompt templates in editor workflows

## V4: Team Learning Platform

- private team-level behavior insights
- manager dashboards for coaching outcomes
- policy packs for responsible AI engineering habits
