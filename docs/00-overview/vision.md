# Vision

AI Dev Coach becomes the standard learning guardrail for developers who code with AI.

## Near-Term Vision (V1)

A Chrome extension that improves prompt quality and flags risky habits.

## Mid-Term Vision (V2)

A cross-tool coaching layer for browser and IDE workflows.

## Long-Term Vision (V3+)

A team-level learning platform that tracks AI usage quality and supports engineering growth, not only productivity output.
